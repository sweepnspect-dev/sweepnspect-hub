🔔 Notification System — Dual Path

Edge Path
24/7
Cloudflare Worker → Resend → Tawk
Hub Path
Backup
Node.js → Nodemailer → Tawk

Flow

Event arrives (FB/Tawk/Form/Ticket)
  │
  ├── Worker classifies priority (critical/high/normal/low)
  │
  ├── If critical or high:
  │   ├── Check KV cooldown (_cooldown:{type}, 5 min)
  │   ├── Build email (emoji + label + context)
  │   ├── POST to Resend API (Bearer token)
  │   ├── Resend sends to tickets@sweepnspect.p.tawk.email
  │   ├── Tawk creates ticket → push notification on phone
  │   └── Store cooldown in KV (10 min TTL)
  │
  └── All events stored in KV alerts + legacy events

🔔 Notification System — Dual Path

Edge Path (24/7)
Resend
Worker → Resend API → Tawk ticket → phone push
Hub Path (backup)
SMTP
Node.js → Nodemailer → Tawk ticket

Flow

Event arrives (FB/Tawk/Form/Ticket)
  |
  +-- Worker classifies priority (critical/high/normal/low)
  |
  +-- If critical or high:
  |   +-- Check KV cooldown (_cooldown:{type}, 5 min)
  |   +-- Build email (emoji + label + context)
  |   +-- POST to Resend API (Bearer token)
  |   +-- Resend sends to tickets@sweepnspect.p.tawk.email
  |   +-- Tawk creates ticket -> push notification on phone
  |   +-- Store cooldown in KV (10 min TTL)
  |
  +-- All events stored in KV alerts + legacy events

To unblock Email Workers

  1. Log into Namecheap → Domain List → sweepnspect.com → Nameservers
  2. Change from registrar-servers.com to Cloudflare nameservers
  3. In CF dashboard: add sweepnspect.com → accept free plan → get nameservers
  4. Re-create existing DNS records (A/CNAME for GitHub Pages, MX, TXT/SPF)
  5. Once propagated: enable Email Routing, create Email Worker
